# Fat Cat Gallery

A polished and sleek website for Fat Cat Gallery, showcasing high-end art collections.

## Features

- Responsive design with modern CSS
- Elegant typography using Google Fonts
- Image-heavy gallery section
- Smooth scrolling navigation
- Contact form

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser or run a local server
3. To run locally: `python3 -m http.server 8000` and visit `http://localhost:8000`

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Playfair Display, Roboto)
